Getting Started with Swift

Swift is a powerful and intuitive programming language developed by Apple. Whether you're building apps for iOS, macOS, or writing command-line tools, Swift has you covered.

Why Swift?

  • Modern syntax that is easy to read
  • Memory safety built in
  • Blazing fast performance
  • Open source and cross-platform

Your First Program

Every Swift journey begins with the classic greeting:

import Foundation

func greet(name: String) -> String {
    return "Hello, \(name)! Welcome to Swift."
}

print(greet(name: "World"))
