Disney+ for Xbox & Windows

Joined the Xbox client team following the prototype project, specializing in animation, performance, and UI polish across the Disney+ and Hulu+ apps. I could take a page developed by others and significantly improve its architecture — eliminating UI-blocking data calls, introducing lazy loading, and restructuring XAML for optimal render performance. This work enabled smooth system-standard page transitions and dramatically reduced home screen load times.

Designed and built a custom sprite sheet video player from scratch using low-level UWP rendering APIs, solving a looping video performance problem the native player couldn't handle. This control powered the animated brand tiles seen on the Disney+ home screen — the fireworks, comic-book flips, and other hover animations described in the New York Times' Disney+ launch coverage. The tile edge-fade animation system I built, where tiles smoothly dim as they scroll to the screen boundary, is still in the app today.

Later led a cross-functional initiative bridging Design and Engineering: audited every color and font reference across the Disney+ and Hulu codebases, reconciled them with the official design library, and built a shared Visual Studio style package that let engineering teams pull design updates via Git with no manual work required.
